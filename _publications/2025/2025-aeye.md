abstract: >-
  Developed A-EYE, a real-time visual assistance framework for the visually impaired using Gemini API to minimize cost barriers. Designed a multi-worker architecture with three independent processes capturing and processing frames at 1-second intervals, ensuring continuous information flow with maximum 3-second latency. Implemented a queue-based TTS system that delivers the most recent environmental description to users. To address inherent LLM processing delays, integrated Depth Anything model for immediate emergency detection—triggering instant alerts when obstacles enter within 1-meter range. Developed a calibration mechanism using human body proportions (height-to-arm-length ratio), where users input their height and extend their arm to calibrate distance measurements, compensating for depth estimation inaccuracies. The complete pipeline achieves 1-second capture-to-alert latency for emergency scenarios while maintaining contextual awareness through continuous LLM-based scene understanding.
